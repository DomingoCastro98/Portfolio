# Portfolio de Domingo Castro

Este proyecto es mi portfolio personal como desarrollador web junior. Esta pensado para presentar mi perfil de forma clara, moderna y visual, mostrando tanto mi experiencia como mis proyectos reales.

## Que es este proyecto

Es una web de presentacion profesional donde se puede:
- conocer quien soy y en que estoy especializado
- ver mis habilidades organizadas por areas
- explorar mis repositorios de GitHub en tiempo real
- contactar conmigo desde un formulario

## Que incluye

- Inicio con presentacion y resumen rapido de perfil
- Seccion "Sobre mi" con contexto profesional y formativo
- Seccion de habilidades con porcentaje por tecnologia
- Seccion de proyectos conectada con mi cuenta de GitHub
- Modal para ver informacion ampliada de cada proyecto
- Formulario de contacto para recibir mensajes por correo
- Modo claro y modo oscuro
- Diseno adaptado a movil, tablet y escritorio

## Objetivo

El objetivo de este portfolio es servir como carta de presentacion digital para oportunidades laborales, colaboraciones y proyectos freelance.

## Enfoque del diseño

Se ha buscado un estilo limpio, profesional y cercano, cuidando:
- legibilidad
- contraste
- navegacion sencilla
- experiencia fluida en cualquier dispositivo

## Estado actual

- Proyecto funcional y listo para publicarse
- Formulario de contacto operativo
- Integracion de proyectos desde GitHub activa
